# `fixdif` 宏包

`fixDif`宏包提供了一个用于定义微分算子样式的接口.

基本命令如下:

- `\d`: 既是数学模式下的微分算子"d"，又是文本中的重音下点；
- `\partial`: “修正”偏微分算子命令；
- `\letdif`: 用于定义微分算子的新命令的接口；
- `\mathdif`: 数学模式下的命令，你可以像`\mathop`等命令一样使用。

更多信息请参考 `fixdif-zh-cn.pdf`。 Happy TeXing!
